This is the full catalog of sound tags the **Cloud API** can detect via [Sound Event Detection](/docs/cloud-api/sound-event-detection).

_Current deep learning model:_ **v2.40.0**(Jun 01, 2026)

*   All
*   Emergency
*   Human action
*   Human status
*   Human speech
*   Home context
*   Outdoor context
*   Animal
*   Music

**NOTE:** _Most sound tags are optimized to be recognized by real sound, not the sound played by laptops, smartphones, or speakers._

The Cloud API uses our built-in catalog only. To detect a sound that isn’t in this list, you need the Edge SDK—see [Custom Sound: Sound Events](/docs/edge-sdk/sound-event-detection/custom-sound).
