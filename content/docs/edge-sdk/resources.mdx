_Current deep learning model:_ **v2.40.0**(Jun 01, 2026)

## 1. How to update your deep learning model

*   For Cochl.Sense Edge SDK **v1.2.0 and above**
    
    *   Simply restarting your program to apply the update.
*   For Cochl.Sense Edge SDK **v1.1.0 and below**
    
    *   Remove the model locally.
    *   Run your session again to download the new model.

## 2. Supported architectures

*   x86-64 (e.g., Intel or AMD)
*   AArch64 (e.g., Raspberry Pi3 Model B, Android device)
*   ARMv7 (e.g., Raspberry Pi2)
