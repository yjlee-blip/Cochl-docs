**Cochl.Sense Dashboard** enables users to analyze and monitor sound data detected by Cochl.Sense. Through the Dashboard, you can create new projects, review analytics, gain insights, and configure actions for detected sounds. It provides detailed feedback and valuable information to enhance your sound monitoring experience.

## Step 1: Sign Up

First, sign up for your Cochl.Sense account or log in at the [Cochl.Sense Dashboard](https://dashboard.cochl.ai/) website.

![](/images/docs/1.png)

Once you have logged in, you will see the following page view:

![](/images/docs/2.png)

## Step 2: Add a new project

To create a new project, click the ‘New project’ button in the upper-right corner of the page. You can create either a Cochl.Sense Cloud API project or a Cochl.Sense Edge SDK project.

<Tabs items={["Cochl.Sense Cloud API","Cochl.Sense Edge SDK"]}>

<Tab value={"Cochl.Sense Cloud API"}>

Start by entering your project name and selecting the project type as Cloud API. Then, click the ‘Create project’ button to add your project.

![](/images/docs/3.png)

Once your project is added, click on it to view the details.

![](/images/docs/4.png)

You can find your project key by clicking ‘Copy Project key’ in the upper-right corner or by navigating to the **Settings** tab of the project. Make sure not to share the project key with anyone outside your project or team.

![](/images/docs/4_v2.png)

![](/images/docs/5.png)

Cochl.Sense Edge SDK is only available to companies with a Mutual Non-Disclosure Agreement (MNDA) in place with Cochl. If you’re interested in using Cochl.Sense Edge SDK, please contact us at [support@cochl.ai](mailto:support@cochl.ai).
