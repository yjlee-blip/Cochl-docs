What you need to get Cochl Security up and running.

## Requirements

**Currently supports NX Witness (Network Optix).** Additional VMS platforms will be supported in future updates.

| Item | Required for |
| --- | --- |
| NX Witness VMS (installed & running) | All tiers |
| Cochl plugin | All tiers |
| Cochl-certified hardware | Entry & Standard (paid plans only) |
| Outbound HTTPS to `api.cochl.ai/security` | All tiers |

## Getting started flow

1

Create an account

Sign up at security.cochl.ai and verify your email

2

Start a free trial

14-day trial — use your existing camera setup, 1 channel included

3

Receive installation package

Download link and license file are sent to your registered email

4

Install and connect

Run the installation script and install the NX Witness plugin

5

Monitor your environment

View detected sound events in real time on the cloud dashboard

For detailed installation instructions, a full product manual is provided to customers after subscription confirmation.
